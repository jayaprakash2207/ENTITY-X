## ğŸ”’ Security Design

- **SSRF Protection** â€” All image fetching blocks private IP ranges (`10.x`, `192.168.x`, `127.x`, etc.)
- **Content-Type Validation** â€” Only accepted MIME types: `image/jpeg`, `image/png`, `image/webp`, `image/gif`, `image/bmp`, `image/tiff`
- **Magic-Byte Verification** â€” File headers are checked against declared content type
- **10 MB Cap** â€” Image downloads are hard-limited to prevent memory exhaustion
- **Electron Context Isolation** â€” Renderer and main processes are fully isolated via `contextBridge`
- **No Legal Advice** â€” All legal outputs carry a mandatory disclaimer; the system never accuses or asserts illegality
